SMS Sender App

By Ofer Klein

This application demonstrates how modern technologies can work together to create a functional and scalable SMS sender service.

Technologies Used

Twilio

Handles SMS sending functionality.

PubNub

Provides real-time messaging updates.

RDS

Relational Database Service for managing and storing data.

Angular 19

Frontend framework for creating a responsive user interface.

Ruby on Rails

Backend framework managing the application's logic and API.
