SMS Sender App

A Home Assignment by Ofer Klein

This application is a fully functional SMS Sender that uses the latest technologies to provide reliable and scalable messaging services.

Key Features and Technologies:

  • Twilio: For sending SMS messages.
  • PubNub: For real-time messaging capabilities.
  • RDS: Relational Database Service for data storage.
  • Angular 19: Frontend framework for a modern UI.
  • Ruby on Rails: Backend API and business logic.

Explore the application to understand its architecture and how it seamlessly integrates various services to create a scalable solution.
